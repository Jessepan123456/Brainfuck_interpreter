# Interpreter
A custom-made Brainfuck-style interpreter for fun

## Features
- Memory tape
- Pointer movement('>' '<')
- Cell Mod ('+' '-')
- Input and output (',' '.')
- Loop ('[' ']')
- ASCII and Num support
- Optional debug mode

1. Reaction Game
3. Pattern Memory
4. Survival
5. Puzzle

## How to Run
1. Download
2. Cargo run
